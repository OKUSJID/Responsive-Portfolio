Linux-Shell

This C++ program is a custom shell that mimics basic functionalities of a UNIX shell. It continuously prompts the user for commands, supporting execution with pipes (`|`) for command chaining and redirections (`<`, `>`). The program parses user input into commands and arguments, creates multiple processes using `fork()`, and manages inter-process communication with pipes. It uses `execvp()` for executing commands and `waitpid()` to synchronize the parent process with its children. This project serves as a foundational implementation of a shell, demonstrating core concepts like process creation, command parsing, input/output redirection, and process communication in a UNIX-like environment.
